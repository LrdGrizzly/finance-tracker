FinanceTracker

A local, free-forever investment dashboard for long-term investors. Runs entirely on your own machine — your data never leaves it.

What it is

Value investing engine with a technical timing overlay

Five-layer signal architecture: Secular (CAPE regime gate) → Selection (Graham, Lynch, Magic Formula fundamentals) → Tactical (trend regime) → Timing (RSI, MACD, Fibonacci confluence) → Risk (margin of safety, volatility-scaled sizing). Hybrid gate + score: hard dealbreakers block, everything else weighs into a 0–100 composite.

Features

Ticker Monitor

Live quote, 12 value-investing fundamentals, strategy-fit deviation table, 5-layer signal verdict, expandable TradingView chart.

Screening pipeline

Magic Formula dual-rank + Graham filter across S&P 500 + EURO STOXX 50 — surfaces ranked value candidates.

Portfolio tracking

Log purchases with market-price prefill, live P/L per position, EUR totals via ECB daily rates.

Calculation transparency

Every signal expands into how it was computed — formula, data considered, sources, honest caveats.

Important

This page is a demo — the app itself runs locally

The dashboard needs its Python backend (live market data, screening, caching), so it can't run on GitHub Pages. To use it, clone the repo and run it on your own machine:

git clone https://github.com/LrdGrizzly/finance-tracker.git

Then follow the README — two commands and it's live at localhost:5001.

Principles
  • Free forever — every data source is free and keyless: Yahoo Finance, ECB, Shiller/multpl, Wikipedia constituents.
  • Local first — holdings and watchlist stay in plain JSON on your machine; market cache is disposable SQLite.
  • Long-term — built for 5+ year horizons, not day trading. BUY ratings are accumulation guidance, not trade triggers.
  • Transparent — no black box: every score traces to named criteria with thresholds and actual values.
